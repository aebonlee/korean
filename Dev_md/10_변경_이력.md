# Korean Pro - 변경 이력

## v1.5.0 (2026-03-26) - 한국어 TTS 전체 사이트 지원

### TTS (Text-to-Speech) 전면 확대
- GlobalTTS 셀렉터 확장: `.expression-table td` 전체 (기존 `td:first-child`만)
- `.dialogue p` 셀렉터 추가 (회화 대화 섹션)
- 한국어 정규식 필터로 비한국어 셀 자동 제외
- Chrome 비동기 음성 로딩 대응 (`voiceschanged` 리스너)
- `ko-KR` 음성 우선 선택 로직 개선
- 버튼/링크 클릭 시 TTS 방지 처리
- 작문 페이지 예시 문장에 `data-tts` 추가
- 에세이 접속사 뱃지에 `data-tts` 추가

### TTS 적용 범위
- 한글 (모음, 자음, 음절, 발음) — 글자 + 예시 단어 모두
- 문법 (조사, 동사활용, 존댓말, 문형) — 모든 한국어 예문
- 회화 (인사, 음식점, 쇼핑, 여행, 일상, 전화) — 표현 + 대화문
- 문화 (K-Drama, K-Pop) — 표현 카드 + 대화 연습
- 작문 (기초문장, 문단, 에세이) — 예시 문장 + 접속사
- 어휘 (기초, 일상, 비즈니스, TOPIK) — 단어 + 예문 카드
