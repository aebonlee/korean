# Korean Pro - 스타일 가이드

## CSS 아키텍처

순수 CSS + CSS Custom Properties 방식을 사용합니다. (Tailwind 미사용)

### CSS 파일 임포트 순서 (`src/index.css`)

```css
@import './styles/base.css';        /* 1. 변수, 리셋, 기본 타이포그래피 */
@import './styles/animations.css';  /* 2. 키프레임, AOS 애니메이션 */
@import './styles/dark-mode.css';   /* 3. 다크 모드 오버라이드 */
@import './styles/navbar.css';      /* 4. 네비게이션 바 */
@import './styles/hero.css';        /* 5. 히어로 섹션, 커리큘럼, 피처 */
@import './styles/footer.css';      /* 6. 푸터 */
@import './styles/site.css';        /* 7. 페이지 공용 (헤더, 레슨, 대화) */
@import './styles/auth.css';        /* 8. 로그인, 소셜 로그인 */
@import './styles/chatbot.css';     /* 9. AI 챗봇 */
@import './styles/vocabulary.css';  /* 10. 어휘 카드, 플래시카드 */
@import './styles/speech.css';      /* 11. 발음 연습 */
@import './styles/responsive.css';  /* 12. 반응형 (모바일, 태블릿) */
```

## 디자인 토큰 (CSS Custom Properties)

### 색상 변수

```css
/* 기본 컬러 (Dark Blue) */
--primary: #1A365D;
--primary-dark: #0F2B46;
--primary-light: #2C5282;
--primary-bg: rgba(26, 54, 93, 0.08);
--primary-bg-strong: rgba(26, 54, 93, 0.15);
--primary-rgb: 26, 54, 93;

/* 그라디언트 */
--gradient-primary: linear-gradient(135deg, var(--primary), var(--primary-light));
--hero-bg: linear-gradient(135deg, #0F2B46 0%, #1A365D 50%, #2C5282 100%);
```

### 5가지 컬러 테마

`<html data-color="blue|red|green|purple|orange">` 속성으로 전환됩니다.

| 테마 | Primary | Dark | Light |
|------|---------|------|-------|
| blue (기본) | #1A365D | #0F2B46 | #2C5282 |
| red | #B91C1C | #991B1B | #DC2626 |
| green | #047857 | #065F46 | #059669 |
| purple | #6D28D9 | #5B21B6 | #7C3AED |
| orange | #B45309 | #92400E | #D97706 |

### 배경/텍스트 변수

```css
--bg-primary: #FFFFFF;      /* 기본 배경 */
--bg-secondary: #F7FAFC;    /* 보조 배경 */
--text-primary: #1A202C;    /* 기본 텍스트 */

```

## 다크 모드

`<html data-theme="dark">` 속성으로 활성화됩니다.

```css
[data-theme='dark'] {
  --bg-primary: #1A202C;
  --bg-secondary: #2D3748;
  --text-primary: #F7FAFC;
  --text-secondary: #CBD5E0;
  --border-primary: #4A5568;
}
```

## 반응형 브레이크포인트

```css
@media (max-width: 1024px) { }  /* 태블릿 */
@media (max-width: 768px) { }   /* 모바일 */
@media (max-width: 480px) { }   /* 소형 모바일 */
```
