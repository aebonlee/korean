# Korean Pro - 컴포넌트 아키텍처

## Context 계층

```
<ThemeProvider>          ← 테마 (다크모드, 컬러)
  <LanguageProvider>     ← 언어 (한국어/영어)
    <AuthProvider>       ← 인증 (Supabase)
      <BrowserRouter>    ← 라우팅
        <PublicLayout>   ← 레이아웃 + Routes
        </PublicLayout>
      </BrowserRouter>
    </AuthProvider>
  </LanguageProvider>
</ThemeProvider>
```

## ThemeContext

**저장 키**: `korean-pro-theme` (다크모드), `korean-pro-color` (컬러)

```js
// 사용법
const { theme, toggleTheme, colorTheme, setColorTheme } = useTheme();
```

**컬러 테마 옵션**:
| 이름 | 색상 코드 |
|------|-----------|
| blue (기본) | #1A365D |
| red | #B91C1C |
| green | #047857 |
| purple | #6D28D9 |
| orange | #B45309 |

## LanguageContext

**저장 키**: `korean-pro-lang`

```js
// 사용법
const { language, toggleLanguage, t } = useLanguage();

// t() 함수: 한국어/영어 텍스트 전환
t('한국어 텍스트', 'English text')
```

## AuthContext (Supabase)

```js
const { user, loading, loginWithGoogle, loginWithKakao, logout } = useAuth();
```

**인증 방식**:
1. Google OAuth: `signInWithOAuth({ provider: 'google' })`
2. Kakao OAuth: `signInWithOAuth({ provider: 'kakao' })`

**Supabase 설정**:
- URL: `VITE_SUPABASE_URL`
- Anon Key: `VITE_SUPABASE_ANON_KEY`
- 변수 prefix: `sb` (sbUrl, sbAnonKey)

## 주요 컴포넌트

### Navbar
- 글래스모피즘 디자인 (backdrop-filter: blur)
- 스크롤 시 배경 변경 (useScrolled)
- 메가 메뉴 (서브 카테고리 드롭다운)
- 모바일 햄버거 메뉴
- 다크 모드 / 언어 / 컬러 토글
- 로고: "K" 아이콘 + "Korean Pro" 텍스트

### SEOHead
- React Helmet 대체 (useEffect 기반)
- `<title>`, `<meta description>`, `<meta og:*>` 동적 설정
- 기본 URL: `https://korean.dreamitbiz.com`
- 기본 OG 이미지: `/og-image.png`

### ChatBot (AI 튜터)
- OpenAI API 연동
- 시스템 프롬프트: 한국어 교사 역할
- localStorage 키: `korean-pro-openai-key`
- 메시지 히스토리 관리

### GlobalTTS (전역 TTS - 클릭→음성)
- `PublicLayout`에서 렌더링 (전역 동작)
- 이벤트 위임: `document.addEventListener('click', handler)`
- CSS 셀렉터 기반 타겟팅:
  - `.expression-table td` (테이블 모든 열)
  - `.expression-card__korean` (표현 카드)
  - `.dialogue-line__korean` (대화 라인)
  - `.dialogue p` (회화 대화 문단)
  - `[data-tts]` (명시적 TTS 속성)
- 한국어 정규식 필터: `/[\u3131-\u3163\uac00-\ud7a3]/` — 비한국어 자동 제외
- `ko-KR` 음성 우선 선택, `voiceschanged` 비동기 로딩 대응
- `.tts-speaking` 클래스로 재생 중 시각 피드백

### useTTS Hook
- `speak(text, options)`: 한국어 TTS 재생 (기본 rate 0.9)
- `speakSlow(text)`: 느린 재생 (rate 0.6)
- `stop()`: 재생 중지
- `isSpeaking`: 재생 상태

### TTSButton (TTS 버튼 컴포넌트)
- Props: `text`, `size ('sm'|'md')`, `showSlow`
- 일반 속도 / 느린 속도 버튼
- SVG 아이콘 (스피커 / 정지)

### SpeechPractice (발음 연습 - STT)
- Web Speech API (`SpeechRecognition`)
- 언어: `ko-KR`
- 텍스트 비교 알고리즘으로 정확도 계산
- 실시간 피드백 제공
