# Korean Pro - 프로젝트 개요

## 프로젝트 정보
- **프로젝트명**: Korean Pro (한국어 프로)
- **URL**: https://korean.dreamitbiz.com
- **GitHub**: https://github.com/aebonlee/korean
- **버전**: 1.5.0
- **설명**: 베트남, 인도, 일본, 영어권 학생을 위한 체계적 한국어 학습 사이트

## 기술 스택
| 항목 | 기술 | 버전 |
|------|------|------|
| UI 프레임워크 | React | 19.x |
| 빌드 도구 | Vite | 8.x |
| 라우팅 | React Router | 7.x |

## 주요 기능
1. **한글 기초**: 자음, 모음, 음절 구조, 발음 규칙
2. **일상 회화**: 인사, 일상생활, 쇼핑, 여행, 음식점, 전화
3. **한국어 문법**: 조사, 동사 활용, 존댓말, 문장 패턴
4. **필수 어휘**: 기초 500, 일상 500, 비즈니스 500, TOPIK 800 (카드 레이아웃)
5. **한국어 작문**: 기초 문장, 문단 작성, 에세이
6. **TOPIK 대비**: 듣기, 읽기, 모의 테스트
7. **한국 문화**: K-드라마, K-팝
8. **AI 한국어 대화**: OpenAI 기반 AI 튜터
9. **발음 연습**: Web Speech API 기반 발음 인식 (STT)
10. **한국어 TTS**: 전체 사이트 한국어 텍스트 클릭 시 음성 재생 (Web Speech Synthesis)

## 인증 시스템
- **Google OAuth**: Supabase를 통한 Google 로그인
- **Kakao OAuth**: Supabase를 통한 카카오 로그인
- Supabase 변수 prefix: `sb` (sbUrl, sbAnonKey)

## 디자인 시스템
- **기본 색상**: Dark Blue (#1A365D)
- **5가지 컬러 테마**: Blue, Red, Green, Purple, Orange
- **다크 모드**: 지원
- **반응형**: 모바일, 태블릿, 데스크톱
- **BEM CSS**: Block__Element--Modifier 명명 규칙
- **CSS Custom Properties**: `--primary`, `--bg-primary` 등
