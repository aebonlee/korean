# Korean Pro - 개발 환경 설정

## 필수 요구사항

- **Node.js**: 18.x 이상
- **npm**: 9.x 이상
- **Git**: 최신 버전

## 초기 설정

### 1. 저장소 클론

```bash
git clone https://github.com/aebonlee/korean.git
cd korean
```

### 2. 의존성 설치

```bash
npm install
```

### 3. 환경 변수 설정

`.env` 파일을 프로젝트 루트에 생성:

```env
VITE_SUPABASE_URL=https://hcmgdztsgjvzcyxyayaj.supabase.co
VITE_SUPABASE_ANON_KEY=your_anon_key_here
```

### 4. OG 이미지 생성

```bash
npm run generate-og
```

### 5. 개발 서버 실행

```bash
npm run dev
```

기본 포트: http://localhost:5173

## 트러블슈팅

### 1. sharp 설치 오류

Windows에서 sharp 설치 실패 시:
```bash
npm install --platform=win32 sharp
```

### 2. Supabase 연결 오류

- `.env` 파일 확인
- Supabase 프로젝트 URL과 Anon Key 확인
- CORS 설정 확인

### 3. GitHub Pages 404

- `CNAME` 파일 확인
- DNS CNAME 레코드 설정 확인
- `vite.config.js`에서 404.html 복사 플러그인 확인
