# Korean Pro - 데이터 구조

## 데이터 파일 위치

| 파일 | 설명 |
|------|------|
| `src/data/vocabulary.js` | 어휘 데이터 |
| `src/data/conversations.js` | 회화 대화 데이터 |
| `src/data/topikData.js` | TOPIK 시험 데이터 |
| `src/config/site.js` | 사이트 설정, 메뉴, 커리큘럼 |

## 어휘 데이터 (`vocabulary.js`)

### 데이터 구조

```js
{
  word: '안녕하세요',       // 한국어 단어
  romanization: 'annyeonghaseyo',  // 로마자 표기
  meaning: 'Hello',        // 영어 의미
  example: '안녕하세요, 저는 미나입니다.', // 예문 (한국어)
  exampleEn: 'Hello, I am Mina.'  // 예문 (영어)
}
```

### 데이터셋

| Export명 | 설명 | 개수 |
|----------|------|------|
| BASIC_VOCAB | 기초 필수 단어 | 20개 |
| DAILY_VOCAB | 일상 단어 | 10개 |
| BUSINESS_VOCAB | 비즈니스 단어 | 8개 |
| TOPIK_VOCAB | TOPIK 단어 | 8개 |

## 회화 데이터 (`conversations.js`)

### 대화 구조

```js
{
  title: '첫 만남 인사',       // 대화 제목 (한국어)
  titleEn: 'First Meeting',   // 대화 제목 (영어)
  lines: [
    {
      speaker: 'A',            // 화자
      ko: '안녕하세요, 저는 김민수입니다.',  // 한국어
      en: 'Hello, I am Kim Minsu.'         // 영어
    },
    // ...
  ],
  vocabulary: [
    { word: '안녕하세요', meaning: 'Hello' },
    // ...
  ]
}
```

### 데이터셋

| Export명 | 설명 |
|----------|------|
| GREETING_DIALOGUES | 인사 & 소개 대화 |
| DAILY_DIALOGUES | 일상생활 대화 |
| SHOPPING_DIALOGUES | 쇼핑 대화 |
| TRAVEL_DIALOGUES | 여행 대화 |
| RESTAURANT_DIALOGUES | 음식점 대화 |
| PHONE_DIALOGUES | 전화 대화 |

## TOPIK 데이터 (`topikData.js`)

### 듣기 문제 구조

```js
{
  question: '다음을 듣고 알맞은 것을 고르세요.',
  audio: '(오디오 설명)',
  options: ['선택지1', '선택지2', '선택지3', '선택지4'],
  answer: 2  // 정답 인덱스 (0-based)
}
```

### 읽기 문제 구조

```js
{
  passage: '한국어 지문 텍스트...',
  questions: [
    {
      question: '이 글의 주제로 알맞은 것은?',
      options: ['선택지1', '선택지2', '선택지3', '선택지4'],
      answer: 1
    }
  ]
}
```

### 데이터셋

| Export명 | 설명 | 개수 |
|----------|------|------|
| TOPIK_LISTENING | 듣기 문제 | 2개 |
| TOPIK_READING | 읽기 지문+문제 | 2개 |
| TOPIK_LEVELS | 레벨 정보 | TOPIK I/II |
