# Korean Pro - 라우팅 구조

## 라우트 맵

모든 라우트는 `src/layouts/PublicLayout.jsx`에서 React Router v7 `<Routes>`로 정의됩니다.
각 페이지는 `React.lazy()`로 동적 임포트되어 코드 분할(Code Splitting)이 적용됩니다.

### 메인 라우트

| 경로 | 컴포넌트 | 설명 |
|------|----------|------|
| `/` | Home | 메인 페이지 |
| `/login` | Login | 로그인 페이지 |
| `/dashboard` | Dashboard | 대시보드 (인증 필요) |
| `/settings` | Settings | 설정 페이지 |
| `/*` | NotFound | 404 페이지 |
